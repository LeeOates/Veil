Enables real-time URL threat checking against Google's database. Free — takes about 2 minutes to set up.

Powers the "Explained by Charlie" AI summary in the popup — a plain-English explanation of what Veil found on the page.

Get a key at console.anthropic.com → API Keys → Create Key. New accounts include free credits.
