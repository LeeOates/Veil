Privacy Policy

Effective date: April 2026 · Oates Technology

Overview

Veil is a browser security extension that helps you identify privacy risks, trackers, dark patterns, and security threats on the websites you visit. This policy explains what data we collect, how we use it, and — most importantly — what we never do with it.

Veil never transmits your browsing history, personal information, or page contents to any server we control. All analysis runs locally in your browser.

Data we process locally

When you visit a page, Veil's content script analyses the page in your browser to detect:

  • Third-party tracker scripts and pixels
  • Cookie counts and cookie-consent banner quality
  • Dark patterns in page text and UI elements
  • Password form safety (HTTP vs HTTPS submission)
  • Fingerprinting scripts
  • AI-generated content signals
  • Phishing and typosquatting indicators

This analysis is performed entirely within your browser. None of this page content is sent to Oates Technology.

External API calls

With your explicit consent (by providing API keys in Settings), Veil may contact:

  • Google Safe Browsing API — sends the current page URL to Google's threat database to check for malware or phishing. Governed by Google's Privacy Policy.
  • Anthropic Claude API — sends anonymised risk signal metadata (score, tracker count, flag list) to generate a plain-English explanation. No page content or personal data is included. Governed by Anthropic's Privacy Policy.
  • HaveIBeenPwned API — sends the current site's domain (not your email or personal details) to check whether that domain has been involved in a known data breach. No account credentials are transmitted. Governed by HIBP's Privacy Policy.

Scan history

Veil stores a short history of scanned sites (up to 60 entries) in Chrome's local extension storage on your device. This data never leaves your browser. You can clear it at any time from the Settings page.

API keys

API keys you enter are stored in Chrome's local extension storage on your device only. They are never transmitted to Oates Technology servers.

Permissions we request

  • storage — to save your settings and scan history locally.
  • tabs / activeTab — to read the URL of the active tab so Veil can analyse it.
  • notifications — to show a Chrome notification when a high-risk page is detected (only if you enable this in Settings).

Children's privacy

Veil does not knowingly collect any personal information from anyone, including children.

Changes to this policy

If this policy changes materially, we will update the effective date above and note the changes in the extension's release notes.
